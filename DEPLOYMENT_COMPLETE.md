## 🆘 IF SOMETHING GOES WRONG

**Issue: Keystore script fails**

- Solution: Run PowerShell as Admin
- Try: `.\scripts\generate-keystore.ps1` again

**Issue: GitHub Actions fails (red X)**

- Solution: Check GitHub Secrets (all 5 there?)
- Try: Regenerate Cloudflare API token

**Issue: APK won't install**

- Solution: Need Android 7.0+
- Try: Enable "Unknown apps" in Settings

**Still stuck?**

- Read: [GITHUB_ACTIONS_MONITOR.md](GITHUB_ACTIONS_MONITOR.md)
- Create: GitHub Issue with error message
